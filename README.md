## 🎯 Project Overview

### What This System Does

| Feature | Description |
|---------|-------------|
| 💳 One-Time Donations | Accepts single donations via card, bank transfer, USSD |
| 🔄 Monthly Giving | Full recurring subscription support via Paystack Plans |
| 🔐 Secure Verification | Server-side payment verification — no fake success |
| 📡 Webhook Processing | Real-time event handling for all Paystack events |
| 📊 Impact Calculator | Live impact display based on donation amount |
| 🎉 Success States | Distinct success flows for one-time vs recurring |
| 📱 Fully Responsive | Works on all devices — mobile first |
| 🌍 Social Sharing | Share donation on Twitter, WhatsApp, Facebook |
